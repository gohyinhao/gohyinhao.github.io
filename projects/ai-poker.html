AI Poker Agent

Description Tools My Role

The AI Poker Agent is an AI that aims to maximize its evaluation function - which in this case refers to the winnings in a game of poker. The agent utilizes a depth-limited minimax search algorithm for tree generation and search. In order to speed up the computation time for each turn, techniques such as bucketing and alpha-beta pruning are employed.

The agent makes decisions based on a weighted evaluation function. These weights are obtained using a variation of local search, pitting variations of our agents against each other, until a maxima is attained, either local or global. A test is then run to check if it is a local maxima. If it is, then the weights are re-initialized and re-trained. The process repeats itself until the global maxima is obtained.

The application was created as part of CS3243 - Introduction to Artificial Intelligence. It is no longer being updated.

The application is adapted from this template and the game of poker is run using PyPokerEngine.
