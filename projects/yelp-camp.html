YelpCamp is a web-based application that simulates an online marketplace for arranging or offering campsites. Note that the application was created for practice and is not actually functional.

Features
  • Account Creation
  • Create/Edit/Delete of Campgrounds

The application was created by following The Web Developer Bootcamp by Colt Steele. It is no longer being updated.
