The Personal Finance Tracker is a finance tracking desktop application to help you effortlessly manage your finances. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).

The application was created as part of CS2103 - Software Engineering. It is no longer being updated.

User Guide

Developer Guide

Features
  • Expense tracking
  • Tracking of debts/bills
  • Budget Creation within specific time frame (days, weeks, months, etc)
  • Statistics to view your spending habits

The application is adapted from AddressBook-Level4 project created by SE-EDU initiative.
