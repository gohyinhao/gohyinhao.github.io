Stock Price Prediction

Description Tools My Role

An AI agent was created and trained to predict stock prices. In our implementation, we used gold prices from the World Gold Council (WGC) which consisted of 10,510 data points. The agent uses neural networks for machine learning in order to give price forecasts of gold over various periods of time - 1-day, 7-day, 30-day, etc.

The agent was created as part of CS3244 - Machine Learning. It is no longer being updated.

Neural Network Optimization
  • Model - Multilayer Perceptron Model (MLP), Long Short-Term Memory Model (LSTM)
  • Activation Function - Tanh, Sigmoid, Rectified Linear Unit (ReLU), Leaky ReLU
  • Optimizer - Stochastic Gradient Descent (SGD), Adaptive Moment Estimation (ADAM)
